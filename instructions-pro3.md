Interactions XR
Documentation des intéractions en mode Immersif (VR/AR) pour l'exploration de graphes.

Contrôleurs
Bouton	Action	Description
Gâchette (Trigger)	Sélectionner	Pointez un nœud et appuyez sur la gâchette pour afficher ses détails.
Grip (Poignée)	Saisir / Tourner	Maintenez le bouton de poignée pour saisir le monde et le faire tourner/déplacer.
Joystick (Thumbstick)	Se déplacer	Utilisez le joystick gauche/droit pour vous déplacer dans l'espace (vol libre).
Bouton A/X	Menu	Ouvre/Ferme le menu contextuel VR.
Locomotion dans l'espace
Puisque le graphe flotte dans le vide, la téléportation classique n'est pas disponible. Nous utilisons le Vol Libre :

Joystick Gauche (Vertical) : Avancer / Reculer dans la direction du regard.
Joystick Gauche (Horizontal) : Pas latéraux (Strafe).
Joystick Droit : Rotation (Snap Turn ou Smooth Turn).
Manipulation du Graphe
Pour observer le graphe sous tous les angles (Rotation & Déplacement) :

Tendez la main vers le vide (pas sur un nœud).
Maintenez le bouton Grip.
Faites pivoter votre poignet ou déplacez votre bras. Le graphe suit votre mouvement comme si vous teniez la scène.
Relâchez pour lâcher.
Manipulation des Nœuds
Pour réorganiser le graphe manuellement :

Pointez un nœud (laser).
Maintenez la Gâchette.
Déplacez le nœud où vous voulez.
Relâchez.
Sélection de Nœud
Visez un nœud (il s'illumine).
Cliquez (pression brève) sur la Gâchette.
Un panneau d'information apparaît devant vous.
Menu Immersif
Un menu virtuel permet d'effectuer des opérations sans quitter l'immersion.

Ouverture : Apparaît automatiquement au démarrage ou via un bouton dédié.
Fonctionnalités :
Dispositions (Layouts) : Changez l'algorithme de disposition du graphe (ex: Sphérique, Kamada-Kawai) en temps réel.
Recentrer : Réinitialise la position de la caméra au centre.
Quitter : Sort du mode VR.
